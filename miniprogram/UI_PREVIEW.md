# 前端界面预览

> 由于微信小程序需要微信开发者工具才能渲染，这里用文字描述和代码结构展示前端设计。

---

## 设计系统

### 色彩方案
```css
--color-primary: #4A90D9        /* 主色调：蓝色 */
--color-primary-light: #E8F0FE  /* 浅蓝背景 */
--color-success: #52C41A        /* 成功：绿色 */
--color-warning: #FAAD14        /* 警告：橙色 */
--color-error: #FF4D4F          /* 错误：红色 */
--color-text-primary: #333333   /* 主文本 */
--color-text-secondary: #666666 /* 次要文本 */
--color-text-hint: #999999      /* 提示文本 */
--color-bg: #F5F7FA             /* 页面背景 */
--color-card-bg: #FFFFFF        /* 卡片背景 */
--color-border: #E8E8E8         /* 边框 */
```

### 圆角和间距
```css
--radius-sm: 8rpx    /* 小圆角 */
--radius-md: 16rpx   /* 中圆角（卡片） */
--radius-lg: 24rpx   /* 大圆角（按钮） */
--radius-full: 9999rpx /* 全圆角（标签） */

--spacing-8: 16rpx   /* 小间距 */
--spacing-16: 32rpx  /* 中间距 */
--spacing-24: 48rpx  /* 大间距 */
```

### 字体大小
```css
--font-xs: 22rpx   /* 小字 */
--font-sm: 24rpx   /* 辅助文字 */
--font-base: 28rpx /* 正文 */
--font-md: 30rpx   /* 中等标题 */
--font-lg: 34rpx   /* 大标题 */
--font-xl: 40rpx   /* 超大标题 */
--font-2xl: 48rpx  /* 巨大标题 */
```

## 组件样式

### 卡片 (Card)
```css
background: #FFFFFF;
border-radius: 16rpx;
padding: 32rpx;
margin-bottom: 32rpx;
box-shadow: 0 2rpx 12rpx rgba(0, 0, 0, 0.04);
```

### 按钮 (Button)
```css
/* 主按钮 */
background: #4A90D9;
color: #FFFFFF;
height: 80rpx;
border-radius: 16rpx;

/* 次要按钮 */
background: #E8F0FE;
color: #4A90D9;

/* 描边按钮 */
background: transparent;
border: 2rpx solid #4A90D9;
color: #4A90D9;
```

### 标签 (Tag)
```css
height: 48rpx;
padding: 0 24rpx;
border-radius: 9999rpx;
font-size: 22rpx;

/* 选中态 */
background: #4A90D9;
color: #FFFFFF;

/* 未选中态 */
background: transparent;
border: 1rpx solid #E8E8E8;
color: #666666;
```

### 悬浮按钮 (FAB)
```css
position: fixed;
bottom: 40rpx;
right: 40rpx;
width: 112rpx;
height: 112rpx;
border-radius: 56rpx;
background: #4A90D9;
color: #FFFFFF;
font-size: 48rpx;
box-shadow: 0 8rpx 24rpx rgba(74, 144, 217, 0.4);
```

## 响应式设计

- 使用rpx单位（750rpx = 屏幕宽度）
- 支持iPhone X+的安全区域
- 横向滚动支持（科目筛选、分类筛选）

## 如何查看实际效果

1. **下载微信开发者工具**
   https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

2. **打开项目**
   - 导入项目目录：`miniprogram/`
   - 选择测试号或自己的AppID

3. **编译预览**
   - 自动编译
   - 点击"预览"生成二维码
   - 手机扫码查看实际效果

详细说明见 `miniprogram/README.md`
