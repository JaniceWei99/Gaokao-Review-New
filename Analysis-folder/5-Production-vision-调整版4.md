

# 上海高考复习助手 - Product Vision Document v4

---

## 🎯 Product Vision

**Vision**: 为上海高考考生家长提供一个"不需要懂学科知识也能用"的复习管理助手，通过极简记录、孩子低压力确认和合规的行动指引，让家长真正参与到孩子的高考复习中。

**Mission**: 降低家长参与高考复习的知识门槛和沟通阻力，通过"粗粒度标记 + 孩子微确认 + 行动话术引导"的三层机制，帮助每个家长——无论是否懂学科知识——都能科学地辅助孩子。

**核心设计哲学**:

> 不要把最困难的部分扔给用户。家长不是教研员，孩子不是项目汇报人。产品要在合规框架内，用技术跨越家长的知识鸿沟和亲子沟通的障碍。

---

## ⚠️ 合规与监管边界

### 教育领域监管要求

教育领域监管严格，2025年标志着教育智能化进入"深度应用与系统规范并举"的新阶段，产品必须在"育人导向、安全可控"原则下严格运行。

#### 🚫 绝对不能做的功能

- **学科类培训**: 不提供任何形式的学科辅导内容
- **拍照搜题给答案**: 不提供题目答案或解题过程
- **在线课程/视频教学**: 不提供任何教学内容
- **题库练习**: 不提供习题和练习功能
- **作业批改**: 不提供作业批改服务
- **名师对接/课程导流**: 不做付费教师撮合平台

#### ✅ 产品定位：家长管理工具

- **核心定位**: "家长管理工具"而非"学习辅导工具"
- **功能边界**: 记录、追踪、提醒、沟通话术引导，不涉及教学内容
- **价值主张**: 帮助家长了解和管理孩子学习状况，降低参与门槛

#### ⚠️ AI功能合规边界（Phase 3+）

- ✅ **可以做**:
  - 高考政策、时间节点、报名流程解读
  - 通用学习方法建议（如"间隔重复法""错题整理法"）
  - 心理调节建议（如"考前焦虑如何缓解"）
  - 亲子沟通话术生成（如"怎么和孩子讨论薄弱点而不引发冲突"）
  - 志愿填报流程指导
  - 基于家长标记数据的优先级排序和行动建议

- ❌ **不能做**:
  - 具体题目的解答
  - 学科知识点讲解（如"倍角公式怎么推导"）
  - 考试预测和押题
  - 定向推荐付费学科培训课程或教师
  - 任何涉及学科培训的内容

- ⚠️ **灰色地带（需法律确认后再上线）**:
  - 拍照识别知识点类别（不给答案）——需确认是否属于"拍题服务"监管范围
  - 推荐公开免费学习资源链接——需确认是否属于"导流学科培训"

## 🎯 市场策略

### 初期定位：上海高考家长工具

- **目标市场**: 上海高三考生家长（每年约5万考生，高中存量约15万）
- **切入点**: 极简薄弱点标记 + 亲子沟通辅助 + 高考里程碑管理
- **核心价值**: 让不懂学科知识的家长也能有效参与孩子的高考复习

### 扩展路径

```
Phase 1：上海高三家长工具（验证核心价值和交互机制）
   ↓ 核心假设验证成功
Phase 2：增强功能，提高用户粘性
   ↓ 付费模型跑通
Phase 3：扩展到浙江、北京等新高考省份
   ↓ 规模化
Phase 4：覆盖全国新高考省份
```

### 底层架构可配置化（为扩展准备）

| 可配置模块 | 说明 |
|-----------|------|
| 科目体系 | 各省高考模式（上海3+3 / 广东3+1+2 等） |
| 知识点树 | 各省教材版本、考纲差异 |
| 里程碑节点 | 各省考试时间、志愿填报时间 |
| 分值体系 | 上海660分 / 全国卷750分 等 |

## 👥 User Personas

### 核心用户：高三考生家长

- **年龄**: 40-50岁
- **特征**:
  - 关心孩子学习，但工作繁忙
  - **毕业20多年，已不记得高中学科细节**
  - 想帮忙但不知道从何入手
  - 会定期查看孩子的试卷和成绩，但**看不懂大部分错题**
  - 愿意为孩子的教育投入时间和金钱
  - 熟练使用微信，但不擅长复杂App
  - 在家长群中活跃，会分享有用的工具
- **真实痛点**:
  - 看到孩子错题但**自己看不懂**，不知道背后是什么知识点
  - 想和孩子沟通学习情况，但**不知道怎么问才不会引起反感**
  - 不清楚学校现在的复习进度和计划
  - 对高考重要时间节点和准备要求不清楚
  - **知道孩子某科薄弱，但不知道接下来能做什么**
- **行为模式**:
  - 晚上或周末和孩子沟通学习情况（通常很简短）
  - 定期参加家长会，了解学校情况
  - 会收集和保存孩子的试卷
  - 愿意花时间记录，但**认知负担必须极低**
  - 在家长群中活跃，会分享有用的工具

### 次要用户：高三考生

- **角色**: 低压力确认家长的标记（可选，不强制）
- **参与度**: 极低，最多每周花1-2分钟点选确认
- **心理边界**: 不愿意被"监控"，需要感受到"家长在支持我"而非"家长在检查我"
- **隐私控制**: 可设置部分信息对家长"不可见"

## 🚀 Core Features

### 0. 选科配置（注册必选）⭐ 前置功能

> **上海高考采用"3+3"模式**：语文、数学、外语为必考科目（每门150分），从物理、化学、生命科学、政治、历史、地理6门中自选3门作为等级考科目（每门70分），总分660分。

- **选科引导**: 新用户注册时必须选择孩子的3门等级考科目
- **动态适配**: 选科后，系统只展示对应的6个科目
- **支持修改**: 选科有变可随时调整（保留已有标记数据）

| 类别 | 科目 | 分值 | 是否必选 |
|------|------|------|----------|
| 必考（大三门） | 语文 | 150分 | ✅ 自动包含 |
| 必考（大三门） | 数学 | 150分 | ✅ 自动包含 |
| 必考（大三门） | 外语 | 150分 | ✅ 自动包含 |
| 选考（小三门） | 物理/化学/生命科学/政治/历史/地理（6选3） | 各70分 | 用户选择 |

### 1. 三层薄弱点标记机制 ⭐⭐ 核心功能

> **设计原则：不要把最困难的工作扔给最不专业的人。**

#### 第一层：家长粗粒度标记（核心，必须极简）

**家长只需标记到二级分类（章节/模块级别），不需要识别具体知识点。**

```
操作流程：
选科目（如：数学）
  → 选章节（如：三角函数）   ← 只到这一步！
  → 选程度（😟有点薄弱 / 😰比较薄弱 / 😱很薄弱）
  → 可选：加一句备注（如："孩子说公式搞混了"）
  → 完成！

耗时：< 30秒
认知负担：极低（家长只需知道"哪个章节"，不需要知道"哪个公式"）
```

**为什么这样设计？**

| 旧方案（v3） | 新方案（v4） |
|-------------|-------------|
| 要求标记到三级知识点（如"倍角公式"） | 只需标记到二级章节（如"三角函数"） |
| 家长需要懂学科知识才能标记 | 家长只需转述孩子说的"哪个章节难" |
| 标记准确率低 → 分析不可靠 | 标记准确率高 → 分析有参考价值 |
| 认知负担：高 | 认知负担：极低 |

#### 第二层：孩子微确认（可选，零压力）

家长标记后，系统自动生成一张**确认卡片**，通过微信发送给孩子：

```
┌────────────────────────────────────────────┐
│ 📝 妈妈记录了一条学习笔记                   │
│                                            │
│ 数学 - 三角函数 标记为"比较薄弱"            │
│                                            │
│ 方便的话帮忙选一下具体是哪部分？             │
│ □ 正弦余弦基础   □ 倍角/半角公式            │
│ □ 和差化积       □ 三角函数图像             │
│ □ 其他:______                              │
│                                            │
│ （不选也没关系，妈妈只是想更好地了解你 ❤️）  │
└────────────────────────────────────────────┘
```

**设计要点**：
- 措辞温暖，不是"检查通知"，是"支持卡片"
- 选项是多选checkbox，点一下就行，**耗时<10秒**
- 明确标注"不选也没关系"——**零压力**
- 如果孩子选了，数据自动细化到三级知识点
- 如果孩子没选，二级标记依然有效

#### 第三层：AI辅助识别（Phase 3+，需确认合规性）

```
家长拍照试卷
  → AI识别题目所属的知识点类别（不给答案、不讲解）
  → 输出："这道题主要涉及【数学-解析几何】和【数学-二次方程】"
  → 家长确认 → 自动标记

⚠️ 此功能需法律团队确认合规性后才上线
```

### 2. 亲子沟通辅助 ⭐⭐ 差异化核心功能

> **这是竞品完全没有的功能，也是家长最真实的痛点。**

#### 2a. 沟通话术引导

每次家长完成标记后，系统自动生成一段**沟通建议**：

```
┌────────────────────────────────────────────┐
│ 💬 今晚可以这样和孩子聊：                    │
│                                            │
│ "我看你最近三角函数这块有点吃力，             │
│  是公式太多记不住，还是看到题目               │
│  不知道用哪个公式？                          │
│  如果是公式太多，要不要试试                   │
│  把常用的公式做一张小卡片贴在桌上？"          │
│                                            │
│ 💡 沟通小贴士：                              │
│ 先肯定孩子的努力，再询问具体困难，            │
│ 避免直接问"你怎么这个都不会"                 │
└────────────────────────────────────────────┘
```

**合规性说明**：
- ✅ 这是**沟通方法建议**，不是学科知识讲解
- ✅ "做公式卡片"是**通用学习方法**（如同"做笔记""画思维导图"），不涉及具体学科内容
- ✅ 沟通话术的核心价值是**降低亲子沟通阻力**，而非提供辅导

#### 2b. 周末5分钟复盘引导

每周五推送一张**复盘引导卡片**，引导家长用最简单的方式和孩子沟通：

```
┌────────────────────────────────────────────┐
│ 🗓️ 本周末复盘建议（5分钟就够）              │
│                                            │
│ 找一个轻松的时刻（比如散步或吃饭时），        │
│ 问孩子三个问题：                             │
│                                            │
│ 1️⃣ "这周哪科感觉最吃力？"                  │
│ 2️⃣ "具体是哪个章节或题型？"                 │
│ 3️⃣ "你觉得我能帮你做什么？"                 │
│                                            │
│ 把孩子的回答随手记录在App里就行。             │
│                                            │
│ ❤️ 记住：这不是检查，是陪伴                  │
└────────────────────────────────────────────┘
```

### 3. 行动卡片系统 ⭐⭐ 差异化核心功能

> **解决"诊断有了，药方呢？"的核心问题——在合规范围内给出行动指引。**

#### 3a. 标记后行动卡片

每次标记薄弱点后，自动生成**一张行动卡片**：

```
┌────────────────────────────────────────────┐
│ 📋 行动建议                                 │
│                                            │
│ 📍 数学 - 三角函数（比较薄弱）               │
│ 📅 距离等级考还有47天                        │
│ 🏫 学校下周将复习此章节                      │
│                                            │
│ 👨‍👩‍👧 家长可以做的：                           │
│ 1. 提醒孩子：学校下周要复习三角函数，         │
│    可以提前翻翻教材回顾一下                   │
│ 2. 问问孩子是否需要额外的练习册               │
│    或教辅（由孩子自己选择）                   │

│                                            │
│ ⚡ 一句话：现在不用着急，学校即将复习，        │

```

每个重要时间节点前推送**一张行动卡片**，而非展示完整时间轴：

### 4. 高考倒计时与里程碑

#### 上海高考完整里程碑时间线

```
高三上学期（9月-1月）
├── 9月      开学，进入一轮复习
├── 11月     期中考试
├── 12月     各区一模考试
├── 1月初    合格考（语文、数学等科目合格性考试）
├── 1月中    ⭐ 外语一年两考-第一次 + 春季高考
└── 1月下    春考志愿填报

高三下学期（2月-6月）
├── 3月      各区二模考试
├── 4月      各区三模考试 / 综合评价报名
├── 5月5-6日  ⭐ 等级考（小三门）—— 比高考早一个月！
├── 5月下    三校生高考
├── 6月7-9日  ⭐⭐ 秋季高考（大三门）—— 最终决战
├── 6月下    成绩公布 + 志愿填报
├── 6月28-30日  合格考（补考/其他科目）
└── 7月      录取结果公布
```

#### 功能设计

- **双倒计时**: 同时展示"距离最近考试"和"距离高考"
- **个性化高亮**: 根据选科自动高亮相关考试
- **轻量展示**: 首页只展示**未来30天内**的相关事件，不展示完整时间轴
- **完整时间轴**: 单独页面可查看全年时间轴（可折叠已过去的节点）
- **自定义事件**: 可添加学校考试、课外活动等
- **行动卡片推送**: 每个里程碑前自动推送行动建议（见上方3b）

### 5. 学校复习进度记录

- **简单记录**: 家长会后一键记录"学校接下来要复习什么"
- **与薄弱点联动**: 自动高亮"学校即将复习到的薄弱章节"
- **行动提示**: "学校下周复习三角函数，这正好是孩子的薄弱章节，可以提醒孩子重点听讲"

### 6. 薄弱点分析视图

- **科目分布**: 各科目薄弱章节数量分布（柱状图）
- **程度统计**: 有点薄弱/比较薄弱/很薄弱 的分布
- **改善追踪**: 已标记为"已改善"的章节数量和趋势
- **优先级排序**: 基于 `薄弱程度 × 距离考试天数 × 分值权重` 自动排序

> ⚠️ 不提供任何学科内容讲解，只做**优先级排序和行动建议**

### 7. 简单的成绩记录

- **分科成绩**: 按科目分别录入
- **趋势追踪**: 各科成绩变化趋势（折线图）
- **正向引导**: 成绩提升时自动高亮"进步了！"，薄弱点改善时关联展示"你的关注起作用了"

## 📚 知识点树设计方案

### 核心变化：降低粒度要求

| 版本 | 标记粒度 | 家长认知负担 | 数据准确性 |
|------|----------|-------------|-----------|
| v3 | 三级（倍角公式） | 高——需要懂学科 | 低——经常标错 |
| **v4** | **二级（三角函数）** | **极低——只需转述孩子的话** | **高——章节级别不容易错** |
| v4（孩子确认后） | 三级（倍角公式） | 无——孩子自己选 | 最高 |

### 知识点树结构

**MVP阶段只需要建设到二级分类**，三级分类作为孩子确认的选项。

```
一级（模块）          二级（章节）⬅ 家长标记到此     三级（知识点）⬅ 孩子确认到此
数学-代数          →  函数基础                   →  定义域/值域、单调性、奇偶性...
                     三角函数                   →  正弦余弦、倍角公式、和差化积...
                     数列                       →  等差数列、等比数列、数列求和...
数学-几何          →  立体几何                   →  线面关系、体积计算、空间向量...
                     解析几何                   →  椭圆、双曲线、抛物线...
物理-力学          →  牛顿运动定律               →  第一定律、第二定律、第三定律...
                     能量守恒                   →  动能定理、机械能守恒...
...
```

### 1. 日常沟通场景（最高频）

**晚上孩子回家，吃饭时随口聊两句**

- 家长："今天学校复习啥了？"
- 孩子："三角函数，烦死了"
- 家长："哪里烦？"
- 孩子："公式太多了记不住"
- 家长打开小程序 → 数学 → 三角函数 → 😰比较薄弱 → 备注"公式多记不住" → 完成
- 系统展示沟通话术：*"你可以说：要不要试试把常用公式抄在一张小卡片上？有时候写一遍比看十遍管用"*
- **耗时：30秒。家长不需要知道"倍角公式"是什么。**

### 2. 试卷分析场景（周末）

**周末翻看孩子试卷，看到好几道红叉**

- 家长看不懂题目，但看到试卷上老师批注"立体几何-3分""解析几何-5分"
- 家长打开小程序 → 数学 → 立体几何 → 😱很薄弱 → 解析几何 → 😰比较薄弱
- 系统自动发送确认卡片到孩子微信
- 孩子看到后点选了"线面关系""椭圆方程"
- **家长耗时：45秒。孩子耗时：10秒。**

### 3. 家长会后场景

**参加学校家长会**

- 老师：接下来两周重点复习解析几何
- 家长打开小程序 → 学校复习进度 → 记录"下两周重点：解析几何"
- 系统自动提示：💡"解析几何是孩子标记的薄弱章节，学校即将复习，可以提醒孩子重点听讲"

### 4. 收到行动卡片场景

**周五晚上收到推送**

- 推送内容：*"本周末复盘建议：找个轻松的时刻问孩子三个问题..."*
- 周六散步时问了孩子，孩子说化学有机化学部分有点吃力
- 家长打开小程序 → 化学 → 有机化学 → 😟有点薄弱 → 完成

### 5. 考前场景

**距离等级考30天**

- 收到里程碑行动卡片："孩子选考的物理、化学、历史将在5月5-6日考试..."
- 查看薄弱点分析 → 优先级排序显示"化学-有机化学"排第一
- 行动卡片建议："和孩子确认有机化学是否已跟上学校复习进度"

## 💰 付费策略

### 家长付费心理分析

#### 付费动机

1. **焦虑驱动**: 高考压力大，愿意为任何可能有帮助的工具付费
2. **效果导向**: 看到实际效果（薄弱点在减少，沟通更顺畅）
3. **省心价值**: 不用自己想"怎么问""怎么帮"，产品直接给话术和行动指引
4. **安心感**: 感觉"做了该做的事"，减轻焦虑

#### 付费阻力

1. **效果不确定**: 不确定是否真的有用 → 用7天免费试用解决
2. **使用门槛**: 担心太复杂 → 用极简设计解决
3. **隐私顾虑**: 担心数据被滥用 → 明确隐私政策
4. **替代方案**: 微信群+Excel → 用沟通话术和行动卡片创造增量价值

## 🔍 风险与应对

### 1. 合规风险 🔴

- **风险**: 功能边界模糊，可能被认定为学科培训
- **应对**: 严格定位"家长管理工具"，不涉及任何教学内容。沟通话术仅提供沟通方法和通用学习方法建议。AI功能边界严格设计，每次更新前进行合规审查。Phase 3的AI拍照识别功能需法律确认后才上线

### 2. 核心假设风险 🔴

- **风险**: 即使降低到二级粒度，家长仍可能不愿意坚持标记
- **应对**: Phase 1上线前用微信群模拟验证。设定明确的Go/No-Go指标：7天
