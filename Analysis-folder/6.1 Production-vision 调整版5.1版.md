## 🎯 Product Vision

**Vision**: 为上海高中生家长提供一个"不需要懂学科知识也能用"的备考陪伴助手，通过里程碑驱动的行动指引、每日励志金句、错题拍照归档和成长记录，让家长从高一到高三**全程陪伴、不焦虑、不缺席、不越界**。

**Mission**: 降低家长参与孩子学习的知识门槛和心理负担，通过"里程碑节点提醒 + 具体行动建议 + 每日金句 + 错题归档 + 成长记录"的五层机制，帮助每个家长——无论是否懂学科知识——都能在正确的时间做正确的事，并留下孩子成长的珍贵轨迹。

**核心设计哲学**:
> 不要把最困难的部分扔给用户。家长不是教研员，孩子不是项目汇报人。产品要做的不是让家长"记录和追踪薄弱点"，而是在关键节点**直接告诉家长该做什么**，用金句缓解全家焦虑，用错题本帮家长留存学习痕迹，用成长册记录每一个值得骄傲的瞬间。

> **v5.1 扩展定位**：从"仅服务高三"扩展为"高一到高三全程陪伴"。高一用户以成绩记录、错题归档、成长记录为主；高二开始选科后，逐步启用里程碑和行动建议；高三全功能激活。

- **题库练习**: 不提供习题和练习功能
- **作业批改**: 不提供作业批改服务
- **名师对接/课程导流**: 不做付费教师撮合平台

#### ✅ 产品定位：家长备考陪伴工具
- **核心定位**: "家长备考陪伴工具"——不是学习辅导工具，也不是数据追踪工具
- **功能边界**: 里程碑提醒、行动建议、金句陪伴、知识点浏览、错题拍照归档、成长记录，不涉及教学内容
- **价值主张**: 在正确的时间告诉家长该做什么，帮家长留存孩子的学习痕迹和成长轨迹

#### ⚠️ AI功能合规边界（Phase 3+）
- ✅ **可以做**:
  - 高考政策、时间节点、报名流程解读
  - 通用学习方法建议（如"间隔重复法""错题整理法"）
  - 心理调节建议（如"考前焦虑如何缓解"）
  - 志愿填报流程指导
  - 基于里程碑节点的个性化行动建议

- ⚠️ **灰色地带（需法律确认后再上线）**:
  - 拍照识别知识点类别（不给答案）——需确认是否属于"拍题服务"监管范围
  - 推荐公开免费学习资源链接——需确认是否属于"导流学科培训"

### 1. 里程碑驱动的行动提醒系统 ⭐⭐⭐ 核心功能

> 提醒卡片**完全由里程碑节点自动触发**，家长不需要"打卡"或"记录薄弱点"。

#### 里程碑时间线（按年级适配）

**高三完整里程碑**：
```
高三上学期（9月-1月）
├── 9月      开学，进入一轮复习
├── 11月     期中考试
├── 12月     各区一模考试
├── 1月初    合格考
├── 1月上旬   ⭐ 春季高考（含外语一考）
└── 1月下    春考志愿填报

高三下学期（2月-6月）
├── 3月      各区二模考试
├── 4月      各区三模考试 / 综合评价报名
├── 5月上旬   ⭐ 等级考（小三门）
├── 6月7-9日  ⭐⭐ 秋季高考（大三门）
├── 6月下    成绩公布 + 志愿填报
└── 7月      录取结果公布
```

**高一/高二通用里程碑**：
```
├── 11月     期中考试
├── 1月      期末考试 / 合格考
├── 4月      期中考试
├── 6月      期末考试 / 合格考
└── 高二上学期  选科确认窗口
```

> ⚠️ 具体考试日期以上海市教育考试院官网公告为准，产品内需设计动态更新能力。

每个里程碑触发**两次提醒**：

| 触发时机 | 提醒类型 | 内容侧重 |

| **里程碑前15天** | 📋 准备期行动卡片 | 提前准备：该买什么、该打印什么、该关注什么 |
| **里程碑前3天** | ⚡ 临门行动卡片 | 最后检查：证件、文具、作息、情绪支持 |

**早上起床，看一眼手机桌面**
- 桌面小组件显示倒计时+今日金句
- 家长看一眼，心里踏实，不需要打开App
- **耗时：2秒**

### 2. 收到里程碑提醒场景

**距离等级考15天，收到推送**
- 点开查看行动卡片 → 看到具体行动清单
- 周末去文具店买了备用文具
- 打开知识点树查看孩子选考科目的章节 → 网上搜了核心公式汇总打印出来

- **耗时：5分钟**

### 3. 错题拍照场景（周末）

**周末翻看孩子试卷，看到几道红叉**

- **耗时：1分钟**

### 5. 成绩录入场景

- 打开小程序 → 成绩记录 → 输入各科分数和满分
- 系统自动对比上次月考
- 弹出提示："🎉 数学进步了21个百分点！要记录到成长册吗？"
- 点击"记录进步" → 自动生成一条成长记录
- **耗时：2分钟**

### 6. 家长会后场景

**参加学校家长会**
- 老师：接下来两周重点复习解析几何
- 家长打开小程序 → 学校复习进度 → 记录"下两周重点：解析几何"
- 系统提示：💡"解析几何相关错题有5道，可提醒孩子复习前翻一下错题本"
- **耗时：30秒**

### 7. 考前场景

**距离高考3天，收到推送**
- 点开行动卡片 → 检查证件、确认路线、准备文具、饮食建议

### 8. 分享金句场景

**看到一句特别好的金句**
- 点击"分享" → 生成精美图片卡片

- 其他家长扫码体验 → 自然获客

### 9. 高考结束后场景

### 家长付费心理分析

#### 付费动机
1. **安心感**: 感觉"做了该做的事"，每个节点都有准备
2. **省心价值**: 不用自己记时间节点、想该做什么，系统直接推送
3. **情绪价值**: 每日金句缓解焦虑，小组件带来仪式感
4. **实用价值**: 行动建议具体可执行（打印资料、准备文具）
5. **记录价值**: 错题本和成长册是长期积累的数据资产，越用越有价值

#### 付费阻力

## 🔍 风险与应对

### 1. 合规风险 🔴
- **风险**: 功能边界模糊，可能被认定为学科培训
- **应对**: 严格定位"家长备考陪伴工具"，不涉及任何教学内容。错题本只做拍照归档，不做解题/批改。AI功能边界严格设计，每次更新前进行合规审查。Phase 3的AI错题分类功能需法律确认后才上线

### 2. 核心假设风险 🟡
- **风险**: 家长觉得"里程碑+金句"价值不够，不愿付费
- **应对**: Phase 1上线前用微信群模拟验证。设定明确的Go/No-Go指标：7天留存>35% + 金句查看率>50% + 里程碑提醒打开率>60%。如未达标，重新评估核心价值主张

### 3. 错题本替代品风险 🟡
- **风险**: 市场已有巧学/橙果/喵喵等成熟错题本工具，用户可能不愿在本产品中再用错题本
- **应对**: 差异化定位——本产品错题本面向"家长帮孩子归档"，而非"学生自用"。与里程碑行动建议联动（"考前翻一下错题本"）是竞品没有的功能。如错题本使用率低于20%，考虑降低优先级

### 4. 用户生命周期风险 🟡
- **风险**: 高三用户高考结束即流失
- **应对**: 推出"高中全程包"（¥399/高一到高三），提前锁定用户。高考结束后推送"导出成长档案"增加产品纪念价值。引导老用户推荐给高一/高二家长

### 5. 数据隐私风险 🔴
- **风险**: 错题照片和成绩数据是高度敏感信息
- **应对**: 所有图片和数据加密存储，不用于任何商业用途。首屏展示隐私承诺。支持用户一键导出/删除所有数据。不向任何第三方共享用户数据

### 6. 存储成本风险 🟡
- **风险**: 错题拍照+证书拍照会产生大量图片存储成本
- **应对**: 免费版限制存储量（10道错题+5条成长记录）。图片自动压缩。标准版存储成本纳入定价考虑。Phase 3考虑冷热数据分层存储
