# vision
  for a 一个适用于上海的高中考生，帮助他们高考复习，有计划有针对性的复习. 
  Target users are 上海的高中考生，家长也可以帮忙的一款软件或者app. 
  The core problem we're solving is 学校有统一的复习计划，但每个学生有自己的知识薄弱点，在整个高中学习期间，怎么能够针对自己的薄弱点有针对性地加强，整体能够了解
  自己的学习情况，优势劣势科目，并能够有效地在短时间内提高。因为学校已经有很多轮的系统的复习计划，所以这个款产品并不需要去增加大量的题库和视频讲解，就是针对薄弱点加
强对自己的认知，同时家长和学生是一个账号，可以有效地帮助孩子做一些输入的工作，比如错题，成绩录入等，减少孩子在输入方面的工作量，更好全面地了解自己，掌握自己的薄弱环
节和优势学科，帮助最终高考顺利。
