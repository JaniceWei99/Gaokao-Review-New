

# 上海高考复习助手 - Product Vision Document（修正版 v3）

---

## 🎯 Product Vision

**Vision**: 为上海高考考生家长提供一个简单实用的工具，帮助记录和追踪孩子的薄弱知识点，配合学校复习计划进行针对性补充，实现个性化复习管理。

**Mission**: 通过家长手工标记和记录的方式，建立孩子的个性化薄弱点图谱，配合学校复习进度和高考时间节点，让家长能够科学地辅助孩子复习。

---

## ⚠️ 合规与监管边界

### 教育领域监管要求

教育领域监管严格，产品必须避开敏感功能：

#### 🚫 绝对不能做的功能
- **学科类培训**: 不提供任何形式的学科辅导内容
- **拍照搜题给答案**: 不提供题目答案或解题过程
- **在线课程/视频教学**: 不提供任何教学内容
- **题库练习**: 不提供习题和练习功能
- **作业批改**: 不提供作业批改服务

#### ✅ 产品定位：家长管理工具
- **核心定位**: "家长管理工具"而非"学习辅导工具"
- **功能边界**: 记录、追踪、提醒，不涉及教学内容
- **价值主张**: 帮助家长了解和管理孩子学习状况

#### ⚠️ AI顾问功能边界设计（Phase 4）

如果后期增加AI顾问功能，需要严格设计回答边界：

- ✅ **可以回答**:
  - 高考政策、时间节点、报名流程
  - 学习方法建议（通用性，如"间隔重复法""番茄钟"等）
  - 心理调节建议（如"考前焦虑如何缓解"）
  - 家长沟通技巧（如"如何与孩子讨论成绩而不引发冲突"）
  - 志愿填报流程指导
- ❌ **不能回答**:
  - 具体题目的解答
  - 学科知识点讲解（如"倍角公式怎么推导"）
  - 考试预测和押题
  - 任何涉及学科培训的内容

## 🔍 竞品分析

### 市场现状

2025年AI高考工具赛道已十分拥挤，但竞品集中在**高考后的志愿填报**和**学生端的学习辅导**两个领域，**备考过程中的家长管理工具**几乎无人涉及。

### 竞品对比矩阵

| 竞品 | 用户规模 | 核心功能 | 目标用户 | 聚焦阶段 | 与本产品关系 |
|------|----------|----------|----------|----------|-------------|
| **作业帮** | 月活9800万+ | 拍照搜题、在线课程 | 学生 | 日常学习 | 非直接竞品（功能无重叠） |
| **小猿搜题** | 月活3000万+ | 拍照搜题、AI讲题 | 学生 | 日常学习 | 非直接竞品（功能无重叠） |
| **QQ浏览器高考频道** | 单季5000万+ | AI志愿填报、高考信息 | 学生+家长 | 高考后志愿填报 | 间接竞品（时间段不同） |
| **夸克高考** | 累计1.2亿+ | 深度搜索、智能选志愿 | 学生+家长 | 高考后志愿填报 | 间接竞品（时间段不同） |
| **教育部"阳光志愿"** | 官方免费 | 免费志愿填报服务 | 学生+家长 | 高考后志愿填报 | 间接竞品（免费替代） |
| **腾讯元宝** | 已有"上海高考"智能体 | 按地区细分AI高考助手 | 学生 | 综合 | 潜在竞品（但面向学生） |
| **科大讯飞AI学习机** | 硬件+软件生态 | 本地化学情分析、个性化学习 | 学生 | 日常学习 | 潜在竞品（学情分析方向） |

```
                   ┌───────────────────┐   ┌───────────────────┐
    面向学生端     │  作业帮、小猿搜题   │   │  夸克、QQ浏览器     │
                   │  科大讯飞学习机     │   │  阳光志愿           │
                   │  （红海，竞争激烈） │   │  （红海，大厂扎堆） │
                   └───────────────────┘   └───────────────────┘
    面向家长端     │                       │   │                     │
                   │  ⭐ 本产品（蓝海）    │   │  部分志愿填报工具   │
                   │  几乎无竞品           │   │  有家长功能         │
                   └───────────────────┘   └───────────────────┘
```

### 差异化壁垒分析

| 差异点 | 为什么大厂短期难复制 |
|--------|---------------------|
| **深度上海本地化** | 大厂追求全国覆盖，不会为单一城市深度定制里程碑体系 |
| **家长视角设计** | 大厂用户增长逻辑围绕学生端，家长端通常只是附属 |
| **备考过程管理** | 大厂聚焦"给答案"（搜题、讲题），而非"管理复习过程" |
| **轻量工具定位** | 大厂倾向做重（硬件+课程+题库），与我们的极简路线相反 |

## 🎯 市场策略

### 初期定位：上海高考家长工具

- **目标市场**: 上海高三考生家长（每年约5万考生，高中存量约15万）
- **切入点**: 薄弱知识点标记 + 学校复习进度记录
- **价值**: 帮助家长了解孩子真实学习状况，配合学校复习

### 扩展路径

```
Phase 1：上海高考家长工具（验证核心价值）
   ↓ 验证成功
Phase 2：增强功能，提高用户粘性
   ↓ 付费模型跑通
Phase 3：扩展到浙江、北京等新高考省份（复制成功模式）
   ↓ 规模化
Phase 4：覆盖全国新高考省份
```

### 扩展前提：底层架构可配置化

| 模块 | 可配置项 | 说明 |
|------|----------|------|
| **科目体系** | 各省高考模式（3+3 / 3+1+2） | 上海3+3，广东3+1+2等 |
| **知识点树** | 各省教材版本、考纲差异 | 按省份加载不同知识点数据 |
| **里程碑节点** | 各省考试时间、志愿填报时间 | 按省份配置时间节点 |
| **分值体系** | 各省总分、单科分值 | 上海660分，全国卷750分等 |

## 👥 User Personas

### 核心用户：高三考生家长

- **年龄**: 40-50岁
- **特征**:
  - 关心孩子学习，但工作繁忙
  - 想帮忙但不知道从何入手
  - 会定期查看孩子的试卷和成绩
  - 愿意为孩子的教育投入时间和金钱
  - 熟练使用微信，但不一定擅长复杂App
- **痛点**:
  - 看到孩子错题但不知道背后的知识点薄弱点
  - 不清楚学校现在的复习进度和计划
  - 想针对性辅导但缺乏系统性记录
  - 对高考重要时间节点和准备要求不清楚
  - 不知道孩子的"3+3"选科组合对应哪些考试和时间
- **行为模式**:
  - 晚上或周末和孩子沟通学习情况
  - 定期参加家长会，了解学校情况
  - 会收集和保存孩子的试卷
  - 愿意花时间记录和整理，但需要简单易用
  - 在家长群中活跃，会分享有用的工具

### 次要用户：高三考生

- **角色**: 提供学习反馈，确认家长的标记
- **参与度**: 较低，主要是被动接受家长的辅助
- **隐私边界**: 部分信息学生可设置为"不可见"（减少心理压力）

## 💡 Value Proposition

**对于家长**：

- 📝 **简单记录**: 用最简单的方式记录孩子薄弱知识点（单次标记<60秒）
- 🗓️ **进度跟踪**: 了解学校复习进度，配合家庭辅导
- ⏰ **时间提醒**: 清晰掌握高考重要时间节点（外语一考、等级考、秋季高考...）
- 🎯 **优先级排序**: 知道"现在最应该关注哪个薄弱点"
- 📊 **可视化**: 一目了然的孩子学习状况变化

**对于学生**：

- 🎯 **精准帮助**: 家长能够针对性地帮助薄弱环节
- ⏰ **减少负担**: 不需要自己整理错题和分析
- 📈 **进度清晰**: 了解自己的复习进度和状况

## 🚀 Core Features

### 1. 选科配置（注册必选） ⭐ 前置功能

> **上海高考采用"3+3"模式**：语文、数学、外语为必考科目（每门150分），从物理、化学、生命科学、政治、历史、地理6门中自选3门作为等级考科目（每门70分），总分660分。

- **选科引导**: 新用户注册时必须选择孩子的3门等级考科目
- **动态适配**: 选科后，系统只展示对应的6个科目知识点树
- **支持修改**: 如果选科有变，可以随时调整（保留已有标记数据）
- **科目展示**:

| 类别 | 科目 | 分值 | 是否必选 |
|------|------|------|----------|
| 必考（大三门） | 语文 | 150分 | ✅ 自动包含 |
| 必考（大三门） | 数学 | 150分 | ✅ 自动包含 |
| 必考（大三门） | 外语 | 150分 | ✅ 自动包含 |
| 选考（小三门） | 物理/化学/生命科学/政治/历史/地理（6选3） | 各70分 | 用户选择 |

### 2. 薄弱知识点标记 ⭐ 核心功能

- **知识点树**: 仅展示已选6个科目的知识点结构化列表
- **手工标记**: 家长根据试卷错题或孩子反馈手工标记薄弱知识点
- **标记等级**: 可以标记薄弱程度（轻微/一般/严重）
- **标记备注**: 可以添加具体备注（如："倍角公式和半角公式搞混"）
- **标记时间**: 自动记录标记时间，追踪变化
- **批量标记**: 一次标记多个相关知识点
- **改善标记**: 当知识点已掌握时，可标记为"已改善"

### 3. 学校复习进度记录

- **进度记录**: 记录学校当前复习的科目和知识点
- **进度时间**: 记录复习进度的时间节点
- **完成状态**: 标记哪些知识点已经复习完成
- **进度对比**: 薄弱点 vs 学校复习进度的对比视图（高亮"学校即将复习到的薄弱点"）
- **复习计划**: 可以记录学校的复习计划安排

### 4. 高考倒计时与里程碑 ⭐

#### 上海高考完整里程碑时间线

```
高三上学期（9月-1月）
├── 9月    开学，进入一轮复习
├── 11月   期中考试
├── 12月   各区一模考试
├── 1月初  合格考（语文、数学等科目合格性考试）
├── 1月中  ⭐ 外语一年两考-第一次 + 春季高考
└── 1月下  春考志愿填报

高三下学期（2月-6月）
├── 3月    各区二模考试
├── 4月    各区三模考试 / 综合评价报名
├── 5月5-6日  ⭐ 等级考（小三门）—— 比高考更早！
├── 5月下  三校生高考
├── 6月7-9日  ⭐⭐ 秋季高考（大三门）—— 最终决战
├── 6月下  成绩公布 + 志愿填报
├── 6月28-30日  合格考（补考/其他科目）
└── 7月    录取结果公布
```

#### 功能设计

- **智能倒计时**: 同时展示"距离最近一次考试"和"距离高考"双倒计时
- **个性化里程碑**: 根据选科自动高亮相关考试（如选了物理，则高亮5月等级考物理时间）
- **自定义里程碑**: 用户可添加学校考试、课外活动等个人事件
- **准备提醒**: 每个里程碑前的准备事项提醒（可设置提前1周/2周/1月）
- **家长准备建议**: 针对每个时间节点的家长行动指南
  - 合格考前：关注孩子基础知识点掌握情况
  - 外语一考前：确认是否需要参加，讨论策略
  - 等级考前：重点关注选考科目薄弱点
  - 模考前：提供良好的复习环境
  - 高考前：心理支持 + 后勤保障
- **时间轴视图**: 可视化展示整个高三重要时间节点
- **里程碑完成追踪**: 标记完成状态，考后记录成绩

### 5. 薄弱点分析视图

- **科目分布**: 各科目薄弱点数量分布（雷达图）
- **薄弱程度统计**: 轻微/一般/严重 的分布饼图
- **时间趋势**: 薄弱点标记和改善的时间变化趋势（折线图）
- **优先级排序**: 基于 `薄弱程度 × 距离考试天数 × 分值权重` 自动排序"最值得优先攻克的知识点"
- **改善追踪**: 已标记为"已改善"的知识点数量和趋势

> ⚠️ 不提供任何学科内容或复习方法建议，只告诉家长"应该先关注什么"

### 6. 简单的成绩记录

- **成绩录入**: 录入重要考试成绩（月考、期中、期末、一模、二模、三模）
- **分科成绩**: 按科目分别录入，与薄弱点标记关联
- **趋势追踪**: 各科成绩变化趋势（折线图）
- **薄弱点关联**: 成绩变化与同期薄弱点标记/改善的对比视图

## 📚 知识点树设计方案

### 设计原则

#### 粒度控制

- **一级分类**: 模块/大类（如：数学-代数、数学-几何）
- **二级分类**: 章节（如：三角函数、立体几何）
- **三级分类**: 具体知识点（如：倍角公式、线面关系）

> MVP阶段先做到三级，四级细分考点根据用户反馈后续迭代。

#### 数量控制

| 类别 | 科目 | 一级 | 二级 | 三级 | 预估总数 |
|------|------|------|------|------|----------|
| 必考 | 语文 | 4-5 | 15-20 | 60-80 | 80-100 |
| 必考 | 数学 | 5-6 | 20-25 | 80-100 | 100-130 |
| 必考 | 外语 | 4-5 | 15-20 | 50-70 | 70-90 |
| 选考 | 物理 | 4-5 | 15-20 | 60-80 | 80-100 |
| 选考 | 化学 | 4-5 | 15-20 | 60-80 | 80-100 |
| 选考 | 生命科学 | 4-5 | 15-20 | 50-70 | 70-90 |
| 选考 | 政治 | 4-5 | 15-20 | 50-70 | 70-90 |
| 选考 | 历史 | 4-5 | 15-20 | 50-70 | 70-90 |
| 选考 | 地理 | 4-5 | 15-20 | 50-70 | 70-90 |

> 注：每位学生实际只使用6个科目（3必考+3选考）的知识点树。

### 知识点数据来源策略

#### 1. 官方权威来源（优先级最高）
- **上海市教育考试院**: 高考考试大纲、考试说明
- **上海市教委**: 课程标准、教学要求
- **各科目课程标准**: 国家课程标准 + 上海地方要求
- **高考真题分析**: 近5年上海高考真题知识点分布

#### 2. 教材体系参考
- **上海高中教材**: 各科目教材目录和章节结构
- **教辅资料**: 权威教辅的知识点梳理
- **学校复习资料**: 重点高中的复习大纲

#### 3. 专家审核机制
- **学科专家**: 邀请上海高中各科目教师审核
- **教研员**: 区级教研员参与知识点体系设计
- **一线教师**: 收集一线教师的反馈和建议

### 知识点维护机制

#### 版本管理
- 每年高考后更新知识点树
- 根据考试大纲变化调整
- 保留历史版本，支持不同届学生

#### 用户反馈机制
- 家长可以提交"找不到知识点"反馈
- 定期分析用户自定义备注，发现缺失知识点
- 建立知识点建议和投票机制

#### 质量保证流程
1. **初始版本**: 基于官方大纲 + 教材体系构建
2. **专家审核**: 学科专家审核和修正
3. **小范围测试**: 邀请10-20个家长和教师试用反馈
4. **持续迭代**: 根据用户反馈持续优化

## 📋 User Journey

### 0. 首次使用场景（Onboarding）⭐

**家长在家长群看到推荐，扫码进入小程序**

```
Step 1：欢迎页
       "帮您系统记录和追踪孩子的高考复习状况"
       ↓
Step 2：选择孩子年级
       高一 / 高二 / 高三
       ↓
Step 3：选择3门等级考科目 ⭐
       从物理、化学、生命科学、政治、历史、地理中选3门
       ↓
Step 4：系统自动生成
       → 6科知识点树
       → 对应的里程碑时间线（高亮选科相关的考试）
       → 距离最近考试的倒计时
       ↓
Step 5：引导标记第一个薄弱知识点
       "试试看：选择一个孩子最近考试中出错的知识点"
       → 选科目 → 选知识点 → 选程度 → 完成！
       → 提示"标记完成！坚持记录，1个月后你会看到完整的分析报告"
       ↓
完成！ 总耗时 < 3分钟
```

### 1. 日常沟通场景

**晚上孩子回家后**

- 家长：今天学校复习了什么内容？有遇到什么困难吗？
- 孩子：今天复习了三角函数，有些题目不太会
- 家长：具体是哪部分不会？是公式记不住还是不会应用？
- 孩子：主要是倍角公式和半角公式搞混了
- 家长打开小程序 → 数学 → 三角函数 → 标记"倍角公式、半角公式"为薄弱点 → 添加备注"公式混淆"
- **耗时：约30-45秒**

### 2. 试卷分析场景

**周末查看孩子试卷**

- 家长看到孩子试卷上的错题
- 家长：这道题考查的是什么知识点？为什么错了？
- 孩子：这道题是考查立体几何的线面关系，我空间想象力不够好
- 家长打开小程序 → 数学 → 立体几何 → 标记"线面关系"为薄弱点 → 标记程度为"严重"
- **耗时：约30-45秒**

### 3. 家长会后场景

**参加学校家长会**

- 老师介绍了接下来的复习计划
- 家长记录：接下来一个月重点复习解析几何
- 家长打开小程序 → 学校复习进度 → 记录"下月重点：解析几何"
- 对比现有薄弱点，发现解析几何确实是薄弱环节 → 提高关注优先级

### 4. 考试前后场景

**重要考试前**

- 小程序提醒：距离一模还有2周
- 家长查看薄弱点分析，查看优先级排序
- 了解哪些薄弱点"最值得优先攻克"
- 考试后记录成绩，观察薄弱点是否有改善趋势

## 💰 付费策略

### 家长付费心理分析

#### 付费动机
1. **焦虑驱动**: 高考压力大，愿意为任何可能有帮助的工具付费
2. **效果导向**: 看到实际效果（薄弱点在改善、记录更清晰）
3. **时间价值**: 节省自己整理和分析的时间
4. **安心感**: 感觉"做了该做的事"，减轻焦虑

#### 付费阻力
1. **效果不确定**: 不确定是否真的有用
2. **使用门槛**: 担心太复杂不会用
3. **隐私顾虑**: 担心孩子学习数据被滥用
4. **替代方案**: 可以用Excel或笔记本记录

### 付费分层设计

#### 免费版（获客 + 验证价值）
- ✅ 全部6科基础薄弱点标记（不限科目数）
- ✅ 高考倒计时（基础版）
- ✅ 基础成绩记录
- ✅ 简单的薄弱点列表视图
- ❌ 无高级分析图表
- ❌ 无优先级排序
- ❌ 无学校进度对比

> 💡 设计逻辑：让家长免费体验核心标记功能，积累数据后才需要付费分析。

#### 标准版 — 月付29.9元 / 年付299元（核心付费）
- ✅ 免费版全部功能
- ✅ 完整里程碑管理 + 家长准备建议
- ✅ 学校复习进度记录 + 薄弱点对比
- ✅ 薄弱点分析图表（雷达图、趋势图）
- ✅ 优先级排序建议
- ✅ 成绩趋势分析

#### 高级版 — 月付49.9元 / 年付499元（增值付费）
- ✅ 标准版全部功能
- ✅ AI顾问（合规范围内：高考政策、学习方法、心理建议）

## 🔍 风险与应对

### 1. 合规风险 🔴
- **风险**: 功能边界模糊，可能被认定为学科培训
- **应对**: 严格定位为"家长管理工具"，不涉及任何教学内容。AI顾问边界严格设计，每次更新前进行合规审查

### 2. 知识点准确性风险 🟡
- **风险**: 知识点树不准确或不完整
- **应对**: 多渠道数据来源 + 专家审核 + 用户反馈"找不到知识点"机制 + 年度更新

### 3. 市场规模风险 🟡
- **风险**: 上海高三家长市场规模有限（每年约5万考生）
- **应对**: Phase 1-2 验证核心价值，Phase 3+ 通过可配置架构扩展到全国新高考省份

### 4. 竞争风险 🟡
- **风险**: 大厂或竞品进入家长端市场
- **应对**: 深耕上海本地化，建立口碑壁垒。大厂倾向全国通用产品，难以为单一城市深度定制

### 5. 家长使用持续性风险 🟡
- **风险**: 家长初期新鲜感过后停止使用
- **应对**: 里程碑提醒保持触达，薄弱点改善追踪提供成就感，考前推送提高回访率

### 6. 隐私与亲子关系风险 🟢
- **风险**: 数据透明化加剧家庭焦虑
- **应对**: 学生可设置部分信息"不可见"，产品引导家长"关注进步而非分数"，家长准备建议以支持而非施压为导向
