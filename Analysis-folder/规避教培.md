# 为什么不能随便做学科教育培训类小程序/App？

这是一个非常好的问题！下面系统地解答。

---

## 📌 核心原因：「双减」政策与审批制

2021年，国家出台了**"双减"政策**（《关于进一步减轻义务教育阶段学生作业负担和校外培训负担的意见》），这是所有限制的根本来源。

### 关键政策要求：

| 要求 | 具体内容 |
|------|----------|
| **审批制** | 对原备案的线上学科类培训机构，改为**审批制**管理，未经许可不得以任何线上方式从事有偿性学科类培训 |
| **办学许可证** | 必须取得属地教育行政部门核发的**《办学许可证》**，仅持有营业执照但无办学许可属于**违规经营** |
| **非营利性** | 面向义务教育阶段学生的学科类线上培训机构，**一律登记为非营利性法人**，不得以营利为目的 |
| **ICP备案** | 取得办学许可证后，还需向省级电信主管部门履行**互联网信息服务备案（ICP备案）** |
| **停止新批** | 各地**不再审批新的**面向义务教育阶段学生的学科类校外培训机构 |
| **资金监管** | 须接入**全国校外教育培训监管与服务综合平台** |

### ⚠️ 违规后果

对于违规的培训平台和应用软件（含App），**网信、公安、电信部门会做好关停、下架等工作**。

---

## 🤔 那作业帮和小猿搜题为什么还在？

这是关键区别所在，它们能存在主要有**三个原因**：

### ① 老牌大机构，拥有合规资质

| 平台 | 背景 |
|------|------|
| **作业帮** | 2015年从**百度**分拆出来，国内用户规模千万级的K12在线教育平台 |
| **小猿搜题** | 母公司猿题库的大股东是**腾讯**，近两轮都进行了投资 |

它们在"双减"政策出台前就已完成备案登记，并在政策收紧后**按要求通过了审批制**，取得了合法的办学许可证。

### ② 核心功能是「工具型」而非直接培训

这是最关键的区别：

```
┌──────────────────────────────────────────────────────┐
│                                                      │
│  📚 纯题库类小程序/App                                │
│  → 上架无特殊资质要求                                 │
│                                                      │
│  🎥 在线视频课程类小程序/App                           │
│  → 需要办学许可证、广播电视节目制作等特殊资质            │
│                                                      │
└──────────────────────────────────────────────────────┘
```

作业帮和小猿搜题最核心的功能是**"拍照搜题"**，属于**学习工具性质**，而非直接销售课程。这在政策定义上是不同的。

### ③ 有强大资本支撑，能满足严苛的合规要求

这类头部企业有足够的资金和法律团队来应对复杂的合规审批流程、资金监管接入、非营利性转型等要求。

## 📋 普通开发者 vs 头部平台 对比

| 项目 | 普通开发者/新机构 | 作业帮/小猿搜题 |
|------|-------------------|----------------|
| 办学许可证 | ❌ 难以获批（新机构已停止审批） | ✅ 已持有 |
| 营利性 | ❌ 学科类须非营利 | ✅ 已完成合规转型 |
| ICP备案 | ❌ 需单独申请 | ✅ 已完成 |
| 核心功能定性 | 若含系统课程即违规 | 以搜题工具为主 |
| 资金监管 | ❌ 须接入全国监管平台 | ✅ 已接入 |
| 资本支撑 | ❌ 缺乏 | ✅ 百度系/腾讯系 |

---

## 💡 对你产品的影响

好消息是，你之前设计的"**上海高考复习助手**"定位为：

> ✅ 家长的管理工具（记录、追踪、提醒）
> ❌ 不是题库、不是在线课程、不是学科培训

这个定位**恰好规避了政策风险**。只要不涉及以下内容，就不需要办学许可证：

| 不碰 ✅ 安全 | 碰了 ❌ 需要资质 |
|-------------|----------------|
| 知识点标记和记录 | 系统性学科课程 |
| 倒计时和时间管理 | 在线直播授课 |
| 成绩趋势可视化 | 拍照搜题给答案 |
| 家长辅助提醒 | 有偿学科辅导 |

> **结论：你不是不能做，而是要看"做什么"。做管理工具完全可以；做学科培训则门槛极高且几乎不可能新入局。你当前的产品定位是安全的。**
