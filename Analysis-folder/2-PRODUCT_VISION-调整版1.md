# 上海高考复习助手 - Product Vision Document (调整版)

## 🎯 Product Vision (调整后)

**Vision**: 为上海高考考生家长提供一个简单实用的工具，帮助记录和追踪孩子的薄弱知识点，配合学校复习计划进行针对性补充，实现个性化复习管理。

**Mission**: 通过家长手工标记和记录的方式，建立孩子的个性化薄弱点图谱，配合学校复习进度和高考时间节点，让家长能够科学地辅助孩子复习。

## 🎯 市场策略调整

### 初期定位：上海高考家长工具
- **目标市场**: 上海高三考生家长
- **切入点**: 薄弱知识点标记 + 学校复习进度记录
- **价值**: 帮助家长了解孩子真实学习状况，配合学校复习

### 扩展路径
1. **第一阶段**: 上海高考家长工具（验证核心价值）
2. **第二阶段**: 增强家校协作功能（扩展使用场景）
3. **第三阶段**: 扩展到全国高考（复制成功模式）

## 👥 User Personas (调整后)

### 核心用户：高三考生家长
- **年龄**: 40-50岁
- **特征**:
  - 关心孩子学习，但工作繁忙
  - 想帮忙但不知道从何入手
  - 会定期查看孩子的试卷和成绩
  - 愿意为孩子的教育投入时间和金钱
- **痛点**:
  - 看到孩子错题但不知道背后的知识点薄弱点
  - 不清楚学校现在的复习进度和计划
  - 想针对性辅导但缺乏系统性记录
  - 对高考重要时间节点和准备要求不清楚
- **行为模式**:
  - 晚上或周末和孩子沟通学习情况
  - 定期参加家长会，了解学校情况
  - 会收集和保存孩子的试卷
  - 愿意花时间记录和整理，但需要简单易用

### 次要用户：高三考生
- **角色**: 提供学习反馈，确认家长的标记
- **参与度**: 较低，主要是被动接受家长的辅助

## 💡 Value Proposition (调整后)

**对于家长**：
- 📝 **简单记录**: 用简单的方式记录孩子薄弱知识点
- 🗓️ **进度跟踪**: 了解学校复习进度，配合家庭辅导
- ⏰ **时间提醒**: 清晰掌握高考重要时间节点
- 🎯 **针对性辅导**: 基于记录的薄弱点进行精准辅导
- 📊 **可视化**: 一目了然的孩子学习状况
- 💰 **付费意愿**: 为孩子的教育投资，看到实际效果

**对于学生**：
- 🎯 **精准帮助**: 家长能够针对性地帮助薄弱环节
- ⏰ **减少负担**: 不需要自己整理错题和分析
- 📈 **进度清晰**: 了解自己的复习进度和状况

## 🚀 Core Features (MVP - 调整后)

### 1. 薄弱知识点标记 ⭐ 核心功能
- **科目体系**: 上海高考6+1科目（语数英+物化生政史地）
- **知识点树**: 每个科目的知识点结构化列表
- **手工标记**: 家长根据试卷错题手工标记薄弱知识点
- **标记等级**: 可以标记薄弱程度（轻微/一般/严重）
- **标记备注**: 可以添加具体备注（如：三角函数恒等变换）
- **标记时间**: 记录标记时间，追踪变化
- **批量标记**: 一次标记多个相关知识点

### 2. 学校复习进度记录
- **进度记录**: 记录学校当前复习的科目和知识点
- **进度时间**: 记录复习进度的时间节点
- **完成状态**: 标记哪些知识点已经复习完成
- **进度对比**: 薄弱点 vs 学校复习进度的对比视图
- **复习计划**: 可以记录学校的复习计划

### 3. 高考倒计时与里程碑 ⭐
- **智能倒计时**: 距离高考及重要考试的实时倒计时
- **上海高考里程碑**: 预设上海高考重要时间节点
  - 合格考（学业水平合格性考试）
  - 等级考（选考科目等级性考试）
  - 春季高考
  - 秋季高考（统一高考）
  - 各区模拟考试（一模、二模、三模）
  - 志愿填报时间
- **自定义里程碑**: 用户可添加个人重要事件
- **准备提醒**: 每个里程碑前的准备事项提醒
- **家长准备建议**: 针对每个时间节点的家长指导

### 4. 薄弱点分析视图
- **科目分布**: 各科目薄弱点数量分布
- **薄弱程度**: 不同程度薄弱点的统计
- **时间趋势**: 薄弱点标记的时间变化趋势
- **优先级建议**: 基于薄弱程度和考试时间的优先级建议
- **复习建议**: 针对薄弱点的复习建议

### 5. 简单的成绩记录
- **成绩录入**: 录入重要考试成绩
- **成绩对比**: 与薄弱点标记进行对比分析
- **趋势追踪**: 成绩变化趋势

## 📋 User Journey (调整后 - 更现实)

### 家长的真实使用流程：

#### 1. 日常沟通场景
**晚上孩子回家后**
- 家长：今天学校复习了什么内容？有遇到什么困难吗？
- 孩子：今天复习了三角函数，有些题目不太会
- 家长：具体是哪部分不会？是公式记不住还是不会应用？
- 孩子：主要是倍角公式和半角公式搞混了
- 家长打开App → 数学科目 → 三角函数 → 标记"倍角公式、半角公式"为薄弱点 → 添加备注"公式混淆"

#### 2. 试卷分析场景
**周末查看孩子试卷**
- 家长看到孩子试卷上的错题
- 家长：这道题考查的是什么知识点？为什么错了？
- 孩子：这道题是考查立体几何的线面关系，我空间想象力不够好
- 家长打开App → 数学科目 → 立体几何 → 标记"线面关系"为薄弱点 → 标记程度为"严重"

#### 3. 家长会后场景
**参加学校家长会**
- 老师介绍了接下来的复习计划
- 家长记录：接下来一个月重点复习解析几何
- 家长打开App → 学校复习进度 → 记录"下月重点：解析几何"
- 对比现有薄弱点，发现解析几何确实是薄弱环节

#### 4. 考试前后场景
**重要考试前**
- App提醒：距离一模还有2周
- 家长查看薄弱点分析，发现还有几个重要薄弱点
- 家长针对这些薄弱点重点辅导孩子
- 考试后记录成绩，观察薄弱点是否改善

## 💰 付费意愿分析

### 家长愿意付费的功能：

#### 1. **薄弱知识点标记系统** (核心付费点)
- **付费原因**: 
  - 能够系统性地记录孩子的学习状况
  - 避免遗忘和混乱
  - 看到孩子的进步轨迹
  - 为辅导提供明确方向
- **定价策略**: 
  - 免费版：基础标记功能（限3个科目）
  - 付费版：全科目 + 高级分析功能
  - 价格：月付19.9元，年付199元

#### 2. **高考里程碑管理** (增值功能)
- **付费原因**:
  - 不了解高考时间线，担心错过重要节点
  - 需要专业的准备建议
  - 希望与其他家长分享经验
- **定价策略**:
  - 基础版：免费查看倒计时
  - 付费版：详细准备建议 + 家长社区

#### 3. **学习报告和趋势分析** (高级功能)
- **付费原因**:
  - 想看到孩子的进步数据
  - 需要专业的分析报告
  - 可以与其他学生进行对比（匿名）
- **定价策略**:
  - 付费版专属功能

#### 4. **专家咨询和辅导建议** (增值服务)
- **付费原因**:
  - 愿意为专业教育建议付费
  - 需要个性化的辅导方案
  - 希望获得专家的答疑解惑
- **定价策略**:
  - 按次收费或会员制

### 付费意愿调研要点：
- 家长最关心的是**效果**和**易用性**
- 价格敏感度：上海家长教育投入意愿较强
- 付费触发点：看到实际效果、孩子成绩改善
- 续费动力：持续的价值体现

## 🎯 产品定位调整

### 核心定位：**家长的高考复习辅助工具**
- 不是学生的学习工具，是家长辅助孩子的工具
- 重点在于**记录**和**追踪**，不是教学和练习
- 强调**简单易用**，降低使用门槛

### 差异化优势：
- 🎯 **专注上海**: 深度了解上海高考体系
- 👨‍👩‍👧 **家长视角**: 从家长角度设计功能
- 📝 **手工标记**: 更精准，更符合实际情况
- 🗓️ **时间管理**: 完整的高考时间线管理
- 💰 **付费清晰**: 明确的价值主张
